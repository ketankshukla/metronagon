# Metronagon Media — Tweet Library

This folder contains ready-to-post tweets for X (Twitter). Each file is one post.

## How to Use

1. Open the tweet file
2. Copy the tweet text
3. Attach the specified image(s) from `public/portfolio/`
4. Post to X

## File Naming Convention

```
[series]-[asset-type]-[number].md
```

## Posting Order (Recommended Launch Sequence)

Post these in order over your first 2 weeks:

| Day | File | Content |
|-----|------|---------|
| Day 1 | `00-pinned-intro.md` | Pin this — your intro tweet |
| Day 1 | `aztec-full-series.md` | The 12-cover showcase (your strongest piece) |
