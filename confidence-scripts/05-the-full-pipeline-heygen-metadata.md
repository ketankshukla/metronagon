## Voice Settings

- **Voice style:** Structured and precise — each layer delivered with clear articulation, like building blocks snapping together
- **Pace:** ~160 words per minute — this script is information-dense, needs a slightly faster clip while maintaining clarity
- **Emphasis words:** "one," "layer one/two/three/four/five," "Full Pipeline," "one partner, one vision, one delivery," "four hundred dollars," "delivered in days"
- **Pauses:** Very brief pauses between layers — just enough for the text overlay to land, then immediately into the next. Longer pause at 0:36 before "Stack them all together"
- **Tone:** Build energy with each layer — start controlled, end powerful
