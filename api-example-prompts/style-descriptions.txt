NOTES
═══════════════════════════════════════

- All prompts are designed for 1024 x 1536 pixel images in sRGB color profile
- Each prompt file contains detailed specifications for art style, composition, color palette, typography, and mood
- Prompts are designed for manual use with AI image generation tools
- No two prompts share the same art style or visual approach
- Styles range across: digital painting, oil painting, watercolor, photography, woodcut, mosaic, textile, sculpture, collage, technical illustration, infographic, calligraphy, and more
